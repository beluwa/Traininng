2. How to make File Geodatabase , Blank Shape files of Blocks, Chocks, Features, Roads, Rivers with Attribute Table ?

  1. Go to Catalog

  2. Make a folder (ABC_Digitization) in the drive (D: or E:)

  3. Right on the folder > New > File Geodatabase (Note: Name should be ABC_NP/MNP/UMNP.gdb)

  4. Rename File Geodatabase as ABC_NP

  5. Right click on ABC_NP >New > Feature Class … >Name (ABC_NP_Block) > Next > Select GCS_WGS_1984>Next>Next>Next>New Feature Class window will appear

3. How to do digitization (fill attributes in the shape files of Block, Chock, Feature, Road and River) using Esri Image?

  1. Load Metropolitan/Sub-Metropolitan/Municipality Shape file

  2. Load Esri Image ( Go to Add > Add Data… > Imagery )

  3. Do correction on Ward Boundaries using topology tool

3. How to do digitization (fill attributes in the shape files of Block, Chock, Feature, Road and River) using Esri Image?

  1. Load Block/Chock/Feature/Road/River Shape file from file geo database

  2. Go to Editor > Start editing > Select the shape file which you want to edit

  3. Go to Create Features > Click Over the shape file > Start digitization (Note: When you want to move over line you must use trace tool) > Attributes > Fill the attribute values
